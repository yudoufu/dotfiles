# yudoufu's dotfiles

## Requirements

The install scripts work on...

* git || curl || wget
* bash

The dotfiles environments included...

* zsh
* neovim

Now it's working Mac only.(Probably work on some Linux distros.)

## Installation

```
curl -L https://github.com/yudoufu/dotfiles/develop/install.sh | bash
```
